Prompt 1: Comprehensive Endpoint Documentation
List Products API
Endpoint
GET /api/products
Purpose

Retrieves a list of products from the database. Supports filtering by category, price range, stock availability, sorting, and pagination.

Authentication

No authentication is required.

Query Parameters
Parameter	Type	Required	Description
category	String	No	Filter products by category
minPrice	Number	No	Minimum product price
maxPrice	Number	No	Maximum product price
sort	String	No	Field used for sorting (default: createdAt)
order	String	No	Sort order: asc or desc
page	Integer	No	Page number (default: 1)
limit	Integer	No	Number of products per page (default: 20)
inStock	Boolean	No	Return only products that are in stock
Success Response

Status Code: 200 OK

{
  "products": [
    {
      "_id": "12345",
      "name": "Wireless Headphones",
      "price": 89.99,
      "category": "electronics",
      "stockQuantity": 45
    }
  ],
  "pagination": {
    "total": 42,
    "page": 1,
    "limit": 20,
    "pages": 3
  }
}
Error Response

Status Code: 500 Internal Server Error

{
  "error": "Server error",
  "message": "Failed to fetch products"
}
Example Request 1
GET /api/products?category=electronics
Example Response
{
  "products": [
    {
      "name": "Wireless Headphones",
      "price": 89.99
    }
  ]
}
Example Request 2
GET /api/products?minPrice=20&maxPrice=100&sort=price&order=asc
Example Response
{
  "products": [
    {
      "name": "Bluetooth Speaker",
      "price": 49.99
    }
  ]
}
Special Considerations
Supports pagination for large datasets.
Supports dynamic sorting.
Supports filtering by category and price range.
Returns only available products when inStock=true.

Prompt 3: Developer Usage Guide
How to Use the Products API
Authentication

No authentication is required for this endpoint.

Sending Requests

Use the GET method:

GET /api/products

Example:

GET /api/products?category=electronics&page=1&limit=10
Understanding Responses

A successful response returns:

Product information
Pagination details
Total product count

Example:

{
  "products": [],
  "pagination": {
    "total": 42,
    "page": 1,
    "limit": 20,
    "pages": 3
  }
}
Handling Errors

If the server encounters an error:

{
  "error": "Server error",
  "message": "Failed to fetch products"
}
JavaScript Example
fetch('https://api.example.com/api/products?category=electronics')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));
