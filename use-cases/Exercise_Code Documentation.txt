Java Task Priority Sorting Algorithm

public static int calculateTaskScore(Task task) {
    Map<TaskPriority, Integer> priorityWeights = Map.of(
        TaskPriority.LOW, 1,
        TaskPriority.MEDIUM, 2,
        TaskPriority.HIGH, 3,
        TaskPriority.URGENT, 4
    );

    int score = priorityWeights.getOrDefault(task.getPriority(), 0) * 10;

    if (task.getDueDate() != null) {
        long daysUntilDue = ChronoUnit.DAYS.between(LocalDateTime.now(), task.getDueDate());

        if (daysUntilDue < 0) {
            score += 30;
        } else if (daysUntilDue == 0) {
            score += 20;
        } else if (daysUntilDue <= 2) {
            score += 15;
        } else if (daysUntilDue <= 7) {
            score += 10;
        }
    }

    if (task.getStatus() == TaskStatus.DONE) {
        score -= 50;
    } else if (task.getStatus() == TaskStatus.REVIEW) {
        score -= 15;
    }

    if (task.getTags().stream().anyMatch(tag ->
            List.of("blocker", "critical", "urgent").contains(tag))) {
        score += 8;
    }

    long daysSinceUpdate = ChronoUnit.DAYS.between(task.getUpdatedAt(), LocalDateTime.now());

    if (daysSinceUpdate < 1) {
        score += 5;
    }

    return score;
}

 Calculates an importance score for a task based on priority,
 due date, status, tags, and recent updates.
 
 The score is used to determine how important a task is when
 sorting tasks by priority.
 
 @param task The task whose importance score will be calculated.
 
 @return An integer representing the task's calculated priority score.
         Higher scores indicate higher importance.
 
 @throws NullPointerException if the task object is null.
 
  Example:
  Task task = new Task("Assignment");
  int score = calculateTaskScore(task);
 
  Important Notes:
  - Overdue tasks receive additional points.
  - Completed tasks receive a large score reduction.
  - Tasks tagged as blocker, critical, or urgent receive bonus points.
  - Recently updated tasks receive additional points.
  - Tasks with higher priority levels start with a higher base score.

Function Purpose

The calculateTaskScore() method calculates an importance score for a task by combining priority level, due date urgency, task status, 
tags, and update history. The score is used to rank tasks so that the most important tasks appear first.

Inputs
Task task – The task object to evaluate.
Output
Integer score representing task importance.
Business Rules
Higher priority tasks receive higher starting scores.
Overdue and near-due tasks receive bonus points.
Completed tasks receive a significant score reduction.
Critical tags increase task importance.
Recently updated tasks receive a small bonus.
